# Graph References

이 폴더에는 graph 공통 참고 자료를 둔다.

현재 참고 문서는 아래와 같다.

- `practical-checklist.md`
  - 실제 graph 문제를 받았을 때 바로 확인하는 빠른 체크리스트
- `modeling-checklist.md`
  - 문제를 정점과 간선으로 어떻게 바꿀지 점검하는 문서
- `implementation-template.md`
  - graph 코드를 어떤 순서로 구현할지 정리한 실전 템플릿
- `common-patterns.md`
  - 자주 나오는 graph 패턴을 간단히 정리한 문서
- `pitfalls.md`
  - graph에서 자주 하는 실수와 함정을 정리한 문서
